# StageFlow - Vercel Deployment Package

## Quick Deploy to Vercel

### Option 1: Vercel CLI (Recommended)
```bash
# Install Vercel CLI
npm install -g vercel

# Navigate to this directory
cd stageflow-deploy

# Deploy
vercel

# Follow prompts:
# - Set up and deploy: Y
# - Scope: Your account
# - Link to existing project: N
# - Project name: stageflow (or your choice)
# - Directory: ./ (current directory)
# - Override settings: N

# For production deployment:
vercel --prod
```

### Option 2: Vercel Web Dashboard
1. Go to https://vercel.com/new
2. Click "Add New..." → "Project"
3. Import this folder or drag & drop the ZIP file
4. Project settings will be auto-detected from vercel.json
5. Click "Deploy"

### Option 3: GitHub Integration
1. Create a new GitHub repository
2. Push this folder to the repository
3. Go to Vercel → "New Project"
4. Import your GitHub repository
5. Vercel will auto-detect settings
6. Click "Deploy"

1. **Chrome Desktop**:
   - Open DevTools → Application → Manifest
   - Check "Service Worker" registration
   - Click install icon in address bar

2. **Mobile (iOS/Android)**:
   - Visit deployed URL

3. **Lighthouse PWA Audit**:

   - Run audit

## Environment Variables

If you need to change the backend URL:

1. **In Vercel Dashboard**:
   - Project Settings → Environment Variables
   - Add: `REACT_APP_BACKEND_URL`
   - Value: Your backend URL
   - Redeploy

2. **For Local Changes**:
   - Create `.env.production` in your source
   - Add: `REACT_APP_BACKEND_URL=https://your-backend.com`
   - Rebuild: `yarn build`

## Troubleshooting

### Routes showing 404
- Ensure `vercel.json` is in the root directory
- Check routes configuration in vercel.json

### PWA not installable
- Verify manifest.json is accessible
- Check service worker registration in DevTools
- Ensure HTTPS (Vercel provides automatically)

### Assets not loading
- Check CORS settings on backend
- Verify REACT_APP_BACKEND_URL is correct
- Check browser console for errors

## Custom Domain

To add a custom domain:
1. Vercel Dashboard → Your Project → Settings → Domains
2. Add your domain (e.g., stageflow.com)
3. Follow DNS configuration instructions
4. Vercel provides automatic HTTPS via Let's Encrypt
